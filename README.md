# 🌱 Análise de Dados no Agro | Agrishow 2026

Este projeto foi desenvolvido a partir de uma imersão prática na Agrishow 2026, com o objetivo de transformar observações de campo em insights estratégicos através da análise de dados.

---

## 🎯 Objetivo

Analisar o posicionamento de empresas, tecnologias e tendências do agronegócio, conectando operação, tecnologia e tomada de decisão baseada em dados.

---

## 🏢 Empresas analisadas

* John Deere
* Case IH
* Massey Ferguson
* Jacto
* Indústrias Colombo
* Komatsu
* Ram Trucks
* Sicoob
* Santander Agro
* TOTVS

---

## 📊 Principais Insights

* Máquinas evoluindo para plataformas tecnológicas
* Forte presença de conectividade e telemetria
* Integração entre operação, crédito e tecnologia
* Foco em eficiência operacional e redução de custos
